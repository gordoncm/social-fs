Our data interaction in social network is inherently different from any classic data interaction module. Most file are rarely deleted or changed (do you remember when was the last time you have removed or edited an image in you facebook account ?) but commonly get appended by multiple users. These differences and many more makes traditional distributed file systems incompatible.
